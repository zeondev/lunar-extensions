# Extensions/Plugins

Here is an example plugin.
```js
  document.getElementById("pluginMenu").innerHTML += `
  <div class="card mt-3">
     <div class="card-header">Plugin Title</div>
     <div class="card-body">
      Plugin Content
     </div>
  </div>
  `
```
Here is how to structure your plugin.

- Plugin
  - plugin.js

Then you can link it in Lunar with the URL!
